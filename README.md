# STM32-PID-Fan-Controller
STM32F407 (Cortex-M4) embedded system: PID-based fan control via LM35 ADC sensing, PWM/MOSFET actuation, GPIO LED zones, SWV/ITM telemetry. HAL drivers support UART, SPI, I2C, CAN. Built with STM32CubeIDE. Skills: Embedded Software Dev, Sensor Integration, HAL Driver Dev, Embedded C.
